# 栈 AlVisStack

说明：提供栈相关操作，使用 `new AlVisStack([])`即可创建数组。

## 1. push(params)

入栈。

**必选参数 1**

`value`：数值

**实例**: `stack.push(1)`

**必选参数 2**

`category`：类别，如 x 轴标签或类别标签

`value`: 值，如 y 轴标签或值标签

**实例**: `stack.push({ month: 'Monday1', sales: 55 })`，其中 month 为 category，sales 为 value。

## 2. pop()

出栈，不提供参数。

## 3. clear()

清空栈。

## 4. size()

返回栈的大小。

## 5. isEmpty

判断栈是否为空。
