# 数组 AlVisArray

说明：提供类 js 数组操作，使用 `new AlVisArray([])`即可创建数组。

## 1. push(params)

新增数据。

**必选参数 1**

`value`：数值
**实例**: `arr.push(1)`

**必选参数 2**

`category`：类别，如 x 轴标签或类别标签

`value`: 值，如 y 轴标签或值标签

**实例**: `arr.push({ month: 'Monday1', sales: 55 })`，其中 month 为 category，sales 为 value.

## 2. pop()

删除数组最后一个数据，不提供参数。

## 3. set(params)

修改数组中的元素。

**必选参数**

`category`：类别，如 x 轴标签或类别标签

`value`: 值，如 y 轴标签或值标签

**实例**: `arr.set({ month: 'Tuesday', sales: 20 })`，其中 month 为 category，sales 为 value.

## 4. insert(params)

任意位置插入元素。

**必选参数**

`category`：类别，如 x 轴标签或类别标签

`value`: 值，如 y 轴标签或值标签

`place`: 元素插入位置

**实例**: `arr.insert({ month: 'Tuesday2', sales: 200 }, 3)`，其中 month 为 category，sales 为 value, 在位置 3 处插入元素.

## 5. delete(params)

删除数组中任意一个元素。

**可选参数**

`category`：类别，如 x 轴标签或类别标签

`value`: 值，如 y 轴标签或值标签

**实例**: `arr.delete({ month: 'Tuesday' })`或`arr.delete({ sales: 200 })`，其中 month 为 category，sales 为 value.

## 6. length

返回数组长度，无参数。

## 7. swap(param1, param2)

交换数组中的两个元素

**必选参数**

`param1｜ param2`：参数 1，仅支持数组下标｜当前元素对象。

**实例**: `arr.swap({ month: 'Tuesday', sales: 13 },{ month: 'Wednesday', sales: 25 })`.

## 8. shift

弹出数组的第一个元素，无参数。

## 9. unshift(params)

队头增加数据。

**必选参数 1**

`value`：数值
**实例**: `arr.unshift(1)`

**必选参数 2**

`category`：类别，如 x 轴标签或类别标签

`value`: 值，如 y 轴标签或值标签

**实例**: `arr.unshift({ month: 'Monday1', sales: 55 })`，其中 month 为 category，sales 为 value.
