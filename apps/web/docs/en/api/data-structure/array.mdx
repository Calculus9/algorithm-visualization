# AlVisArray

Description: Provides js-like array operations. Use `new AlVisArray([])` to create an array.

## 1. push(params)

Add new data.

**Required parameters 1**

`value`：value
**eg**: `arr.push(1)`

**Required parameters 2**

`category`：category label, such as x-axis labels or category labels.

`value`: Values label, such as y-axis labels or value labels.

**eg**: `arr.push({ month: 'Monday1', sales: 55 })`, where month is category and sales is value.

## 2. pop()

Delete the last data in the array without providing parameters.

## 3. set(params)

Modify the elements in an array.

**Required parameters**

`category`：category label, such as x-axis labels or category labels.

`value`: Values label, such as y-axis labels or value labels.

**eg**: `arr.set({ month: 'Tuesday', sales: 20 })`, where month is category and sales is value.

## 4. insert(params)

Insert an element at any position.

**Required parameters**

`category`：category label, such as x-axis labels or category labels.

`value`: Values label, such as y-axis labels or value labels.

`place`: Element insertion position

**eg**: `arr.insert({ month: 'Tuesday2', sales: 200 }, 3)`, where month is category and sales is value, insert element at position 3.

## 5. delete(params)

Delete any element in the array.

**Optional parameters**

`category`：category label, such as x-axis labels or category labels.

`value`: Values label, such as y-axis labels or value labels.

**eg**: `arr.delete({ month: 'Tuesday' })` or `arr.delete({ sales: 200 })`, where month is category and sales is value.

## 6. length

Return the length of array.

## 7. swap(param1, param2)

Swap two elements in an array.

**Required parameters**

`param1｜param2`：Parameter 1 & 2, only supports array subscript | current element object.

**eg**: `arr.swap({ month: 'Tuesday', sales: 13 },{ month: 'Wednesday', sales: 25 })`.

## 8. shift

Pop the first element of an array, no parameters.

## 9. unshift(params)

Add data to the head of the queue.

**Required parameters 1**

`value`: value
**eg**: `arr.unshift(1)`

**Required parameters 2**

`object`：new add object

**eg**: `arr.unshift({ month: 'Monday1', sales: 55 })`, where month is category and sales is value.
