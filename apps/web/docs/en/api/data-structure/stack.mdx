# AlVisStack

Description: Provides stack-related operations. Use `new AlVisStack([])` to create an array.

## 1. push(params)

Push to stack.

**Required parameters 1**

`value`：value

**eg**: `stack.push(1)`

**Required parameters 2**

`category`：category label, such as x-axis labels or category labels.

`value`: Values label, such as y-axis labels or value labels.

**eg**: `stack.push({ month: 'Monday1', sales: 55 })`, where month is category and sales is value.

## 2. pop()

Pop the stack without providing any parameters.

## 3. clear()

Clear the stack.

## 4. size()

Returns the size of the stack.

## 5. isEmpty

Returns whether the stack is empty.
